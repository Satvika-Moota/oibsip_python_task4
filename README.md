**SKYCAST** is a modern weather forecasting desktop application built with **Python and Tkinter**.
It provides real-time weather updates and a 5-day forecast with a premium glass-style user interface.

---

##  Features
* Real-time weather information
* 5-day weather forecast
* Auto-detect city using IP location
* Displays temperature, humidity, wind speed & weather condition
* Modern glassmorphism UI
* Predefined popular Indian cities
* Error handling for invalid city names

##  Technologies Used
* Python
* Tkinter & ttk
* OpenWeatherMap API
* Requests
* Geocoder
* Pillow (PIL)

##  Installation
1. Clone the repository:
```bash
git clone https://github.com/Satvika-Moota/oibsip_python_task4
```

2. Navigate to the project folder:
```bash
cd SkyCast-Weather-App
```

3. Install required libraries:
```bash
pip install requests geocoder pillow
```

4. Add your OpenWeather API key in the code:
```python
API_KEY = "YOUR_API_KEY"
```

5. Run the application:
```bash
python main.py
```

##  How to Use
1. Select a city from the dropdown or use **Auto Detect My City**
2. Click **Get Weather Report**
3. View current weather and 5-day forecast instantly
